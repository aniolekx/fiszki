# Product Context: Fiszki

## 1. Problem Solved

Provides a simple, digital alternative to physical flashcards for learning and memorization. Addresses the need for an accessible and organized way to manage study materials.

## 2. How It Should Work

*   Users register and log in to access their personal flashcard collection.
*   Users can create decks to group flashcards by subject or topic.
*   Within a deck, users can add flashcards with a "front" (e.g., question, term) and a "back" (e.g., answer, definition).
*   Users can browse their decks and the flashcards within them.
*   A basic study mode allows users to review cards one by one, revealing the back after viewing the front.

## 3. User Experience Goals

*   **Intuitive:** Easy to navigate and understand without instructions.
*   **Efficient:** Quick to create and manage decks/cards.
*   **Focused:** Minimal distractions during study sessions.
*   **Reliable:** User data is securely stored and accessible.

*(This context is derived from the project brief and general understanding of flashcard applications.)*
