# Project Brief: Fiszki

## 1. Project Goal

To create a web application for managing and studying flashcards (Fiszki).

## 2. Core Requirements

*   User authentication (login/registration).
*   Ability for users to create, view, edit, and delete flashcard decks.
*   Ability for users to add, view, edit, and delete flashcards within decks.
*   A study mode for reviewing flashcards.

## 3. Target Audience

Individuals looking for a digital tool to create and study flashcards.

## 4. Scope

*   **In Scope:** Basic CRUD operations for users, decks, and flashcards. Simple study interface.
*   **Out of Scope (Initially):** Advanced features like spaced repetition, sharing decks, import/export, mobile app.

## 5. Key Technologies

*   Symfony 6.4+
*   PHP 8.3+
*   Twig
*   Doctrine ORM
*   MySQL
*   Bootstrap 5
*   Webpack Encore
*   Docker
